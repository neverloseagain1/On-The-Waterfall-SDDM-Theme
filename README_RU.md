# SDDM Тема On-The-Waterfall 🦖🌊

Язык: [English](README.md) | [Русский](README_RU.md)

Мрачная, атмосферная тема входа для SDDM в стиле симулятора выживания **The Isle (Legacy)**. Она отличается строгой отзывчивой разметкой, высокопроизводительным нативным размытием левой панели и тщательно подобранной палитрой цветов, вдохновлённой мокрыми скалами, туманными водопадами и дикой природой.

Создана для того, чтобы передать суровую атмосферу выживания среди динозавров при каждом запуске вашей Linux-системы.

---

## ✨ Особенности

- 🎭 **Эстетика игры The Isle:** Холодная, реалистичная палитра из глубоких тёмно-серых скальных тонов и серо-зелёных акцентов.
- 🧪 **Нативный глассморфизм Qt6:** Кастомная панель размытия на базе `Qt5Compat.GraphicalEffects`, которая корректно обрабатывает левую часть фона без сжатия картинки.
- 🎛️ **Кастомный выбор сессий:** Полностью переписанное выпадающее меню (без багов и ломающихся поп-апов) для выбора графических окружений и оконных менеджеров (Hyprland, Plasma, Sway и т.д.).
- 🖋️ **Поддержка шрифтов:** Интегрированный кастомный шрифт `Daneehand Regular` для всех текстовых элементов и полей ввода.
- 🕒 **Часы выживания:** Динамические часы со статусным выводом текущего времени и локализованной даты.
- 👤 **Умное запоминание пользователя:** Автоматически подгружает имя последнего вошедшего в систему пользователя.

## 📂 Установка

### 1. Ручная установка

Склонируйте этот репозиторий или скачайте ZIP-архив, после чего перенесите папку напрямую в системную директорию тем SDDM:

```bash
# Клонирование репозитория
git clone https://github.com/neverloseagain1/On-The-Waterfall-SDDM-Theme

# Перенос папки темы в системный каталог под правильным именем
sudo cp -r On-The-Waterfall-SDDM-Theme/ /usr/share/sddm/themes/On-The-Waterfall
```

### 2. Активация темы

Отредактируйте системный конфигурационный файл SDDM (обычно он находится по пути `/etc/sddm.conf` или `/etc/sddm.conf.d/theme.conf`). Укажите имя темы в секции `[Theme]`:

```ini
[Theme]
Current=On-The-Waterfall
```

---

## ⚙️ Настройка конфигурации

Вы можете легко изменить цвета, шрифты или пути к изображениям, отредактировав файл `theme.conf` внутри папки темы:

```ini
[General]
# Цветовая палитра в формате HEX (Мокрые скалы и Туман)
ColorLight="#D2D7DF"
ColorDark="#1B1E22"
ColorAccent="#5C6F65"
ColorInputBg="#0E1113"

# Настройки шрифта
Font="daneehandregular"
FontSize="20"

# Путь к фоновому изображению
Background="backgrounds/background.jpg"
```

---

## 🛠️ Тестирование без перезагрузки

Вы можете проверить внешний вид и работу темы прямо на рабочем столе, не выходя из системы, выполнив следующую команду в терминале:

```bash
sddm-greeter-qt6 --test-mode --theme /usr/share/sddm/themes/On-The-Waterfall
```

## 📜 Зависимости

Убедитесь, что в вашем дистрибутиве установлен необходимый слой графических эффектов для корректной работы размытия панели в Qt6:
- Для **Arch Linux**: `sudo pacman -S qt6-5compat`
- For **Ubuntu/Debian**: `sudo apt install qml6-module-qt5compat-graphicaleffects`
- For **Fedora**: `sudo dnf install qt6-qt5compat`
